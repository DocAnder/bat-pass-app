## Projeto de Prática com React Native e Expo

Repositório criado para praticar o desenvolvimento de interfaces móveis usando React Native e Expo.

O objetivo da aplicação é gerar uma senha aleatória, permitindo que o usuário utilize o botão GENERATOR para obter senhas e o botão COPY para copiar a senha gerada para a área de transferência. 


1. Estrutura do código com enfoque para:

- Criação de Componentes
- Organização do código
- Gerenciamento de estados
- Envio de propriedades
- Estilização
